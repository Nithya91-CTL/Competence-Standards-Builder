# Competence Standards Builder

Prototype tool for academics to draft competence standards from course or programme information.

## Run locally

```powershell
python app.py 8000
```

Then open `http://127.0.0.1:8000`.

The prototype uses a small Python standard-library web server and `python-docx` for Word downloads. It embeds the reference framing from the June 2026 competence standards note, including descriptors, justification prompts and resources.
